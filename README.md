<ins>****User CRUD API****</ins>  
A RESTful API built using Node.js, Express.js, MongoDB, and Mongoose.

**Features**

- Create User
- Get All Users
- Get User By ID
- Update User
- Delete User

**Technologies Used**

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Postman

**API Endpoints**  

POST /api/users  
GET /api/users   
GET /api/users/:id   
PUT /api/users/:id   
DELETE /api/users/:id
